# Lost Signal — JavaScript Platformer

## Описание
**Lost Signal** — это 2D-игра в жанре платформер, выполненная на **Vanilla JavaScript** с использованием **HTML5 Canvas API**. Игрок управляет энергетическим ядром, которое проходит через три зоны, собирает все фрагменты сигнала, избегает шипов и дронов, активирует портал и восстанавливает систему маяков.

Проект полностью переработан как законченная мини-игра с логикой прогрессии, усложнением уровней, проигрышем и финальным экраном.

## Реализованный функционал
- движение влево/вправо: **A/D** или **←/→**
- прыжок: **W / Space / ↑**
- плавная физика движения и гравитация
- коллизии с платформами
- сбор энергетических фрагментов
- портал, который открывается только после сбора всех фрагментов
- 3 уровня с нарастающей сложностью
- ловушки: шипы и движущиеся дроны
- система жизней
- экран победы и экран поражения
- пауза и рестарт
- сохранение лучшего результата через **localStorage**
- модульная архитектура на ES Modules

## Как запустить проект
### Локально
1. Скачать проект.
2. Открыть папку в **VS Code / VSCodium**.
3. Запустить через **Live Server**.

### Через GitHub Pages
1. Загрузить файлы в репозиторий GitHub.
2. Включить **Settings → Pages → Deploy from a branch**.
3. Выбрать ветку `main` и папку `/root`.

## Идея игры
Игрок управляет энергетическим шаром — ядром системы. На каждом уровне нужно собрать все энергетические фрагменты, открыть портал и пройти в следующую зону. Финальная цель — восстановить потерянный сигнал сети маяков.

## Учебные темы, использованные в проекте
В проекте были применены:
- базовые конструкции JavaScript (`let`, `const`, массивы, функции)
- события клавиатуры (`addEventListener`)
- ООП (классы `Game`, `Player`, `Platform`)
- модули (`import/export`)
- локальное хранилище (`localStorage`)
- игровой цикл на `requestAnimationFrame`
- Canvas API для отрисовки игры

Темы, связанные с формами, API и cookies, в проекте не использовались, так как они не требовались по логике игры.
