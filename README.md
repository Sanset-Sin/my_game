# Lost Signal: Grounds — JavaScript Platformer

## Описание
**Lost Signal: Grounds** — это 2D-игра в жанре платформер на **Vanilla JavaScript** с использованием **HTML5 Canvas API**. Игрок управляет ядром-мячом, катится по длинным участкам твёрдой земли, перепрыгивает обрывы, забирается на блоки, лавки и стены, избегает шипов и летунов и доходит до портала.

Версия переработана с упором на оптимизацию: уменьшена нагрузка на рендер, добавлена отсечка объектов вне экрана, интерфейс стал легче, а переходы между уровнями и после потери жизни продолжаются без повторного нажатия кнопки старта.

## Реализованный функционал
- движение влево/вправо: **A/D** или **←/→**
- прыжок: **W / Space / ↑**
- игровой цикл на `requestAnimationFrame`
- более тяжёлая гравитация и улучшенное падение
- длинные уровни с твёрдой землёй и обрывами
- коллизии со стенами, блоками и осязаемыми скамейками
- шипы на земле и на вертикальных колоннах
- летающие враги
- переход к следующему уровню без обязательного сбора всех бонусов
- автопродолжение после перехода и после потери жизни
- итоговая статистика по каждому уровню и по всей игре
- сохранение лучшего результата через `localStorage`

## Как запустить проект
### Локально
1. Скачать проект.
2. Открыть папку в **VS Code / VSCodium**.
3. Запустить через **Live Server**.

### Через GitHub Pages
1. Загрузить файлы в репозиторий GitHub.
2. Включить **Settings → Pages → Deploy from a branch**.
3. Выбрать ветку `main` и папку `/root`.

## Учебные темы, использованные в проекте
В проекте использованы:
- базовые конструкции JavaScript (`let`, `const`, массивы, функции)
- события клавиатуры (`addEventListener`)
- ООП (классы `Game`, `Player`, `Platform`)
- модули (`import/export`)
- локальное хранилище (`localStorage`)
- Canvas API
- игровой цикл на `requestAnimationFrame`

Формы, cookies и API в проекте не использовались, так как игра полностью локальная.
