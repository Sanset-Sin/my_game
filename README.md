# Lost Signal: Reboot — JavaScript Platformer

## Описание
**Lost Signal: Reboot** — это 2D-игра в жанре платформер на **Vanilla JavaScript** с использованием **HTML5 Canvas API**. Игрок управляет ядром-мячом, проходит три уровня, перепрыгивает дыры в земле, избегает шипов и летающих врагов, собирает бонусы и добирается до портала.

Версия переработана с акцентом на оптимизацию: упрощён рендер, снижена визуальная нагрузка, улучшена физика падения и прыжка, убрано обязательное условие по сбору всех бонусов для перехода между уровнями.

## Реализованный функционал
- движение влево/вправо: **A/D** или **←/→**
- прыжок: **W / Space / ↑**
- игровой цикл на `requestAnimationFrame`
- гравитация, трение и коллизии
- чёткая земля с провалами и отдельными платформами
- стены, на которые можно запрыгивать
- бонусы для итоговой статистики
- переходы между уровнями без обязательного сбора всех бонусов
- шипы, летающие враги, декоративные объекты
- 3 уровня
- система жизней
- итоговая статистика по каждому уровню и по всей игре
- сохранение лучшего результата через `localStorage`

## Как запустить проект
### Локально
1. Скачать проект.
2. Открыть папку в **VS Code / VSCodium**.
3. Запустить через **Live Server**.

### Через GitHub Pages
1. Загрузить файлы в репозиторий GitHub.
2. Включить **Settings → Pages → Deploy from a branch**.
3. Выбрать ветку `main` и папку `/root`.

## Учебные темы, использованные в проекте
В проекте использованы:
- базовые конструкции JavaScript (`let`, `const`, массивы, функции)
- события клавиатуры (`addEventListener`)
- ООП (классы `Game`, `Player`, `Platform`)
- модули (`import/export`)
- локальное хранилище (`localStorage`)
- Canvas API
- игровой цикл на `requestAnimationFrame`

Формы, cookies и API в проекте не использовались, так как игра полностью локальная.
